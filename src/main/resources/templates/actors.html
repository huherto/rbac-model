List of Actors and Goals

Data view user.

  • Find a hunit entry.
  • View a hunit entry.
  • View active hierarchy.
  • View hierarchy active on date.

Data entry user.

  • Submit hunit entry.
  • Add new child.
  • Move to different parent.
  • Decomision.
  • Resubmit hunit entry.

Approver.

  • Certify hunit entry.
  • Reject hunit entry.
  • Change effective end date on submitted entry.

Batch job.

  • Activate hunit entry.
  • Deactivate hunit entry.
  • Expand hunit into derived.

External applications (accessing exposed web services).

  • Return active ancestors for hunit. Given htype, hunit and date.
  • Return complete active hierarchy. Given htype and date.

Data admin user

  • Create new hierarchy type.
